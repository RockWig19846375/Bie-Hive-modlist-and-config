# Bie-Hive-modlist-and-config
Contains required mods and config folder for the Bie Hive

In order to hide the IP as this is a public repository, ask RockWig for the config folder

Installation instructions:
-just add stuff in the mods folder into your mods foleder.
-load Fabric Loader 0.16.7 for Minecraft 1.20.4
-launch the game
-if you properly added the config folder, the initial loading screen should be blue

Joining the server:
-if you installed the config correctly, there should also be an extra button next to Multiplayer to directly connect to the server
-first time joining will ask you to synchronize files, just synchronize configs and reload the game
-you can now play as you like
